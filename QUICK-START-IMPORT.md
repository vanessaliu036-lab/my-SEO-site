## ⚡ 3 分鐘快速導入

### 步驟 1: 打開 Airtable

前往您的 Airtable workspace 並找到 **OCC_Blog_Posts** 表

### 步驟 2: 導入 CSV

1. 點擊表上方的 **+ Add** 按鈕
2. 選擇 **Import CSV**
3. 上傳 `OCC-Volume2-Articles.csv` 文件
4. 點擊 **Import**

### 步驟 3: 驗證

1. 刷新 `/blog` 頁面

## 🆘 常見問題

**Q: 文章不顯示怎麼辦?**  
A: 確保 Status 欄設置為 "Publish"（不要用 "Draft"）

**Q: 我可以編輯文章嗎?**  
A: 是的！在 Airtable 中編輯，變更會自動同步到網站

**Q: 如何添加特色圖片?**  
A: 在 Airtable 的 "Featured Image URL" 欄輸入圖片 URL

**Q: 可以更改文章順序嗎?**  
A: 在 Airtable 中編輯 "Publish Date" - 列表按日期排序
